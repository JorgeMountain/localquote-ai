RESUMEN
Tactio es un MVP SaaS para negocios locales. Permite configurar negocios, FAQs, enlaces,
agenda opcional, chat web, WhatsApp Cloud API, clientes, conversaciones, solicitudes de
cita, cotizaciones y comprobantes de pago.

ARQUITECTURA ACTUAL
- Next.js 16 App Router y React 19.
- Supabase Auth con perfiles platform_admin y business_owner.
- RLS por propietario; el administrador puede consultar todos los negocios.
- El chat público solo lee la ficha comercial segura.
- Las escrituras del chat pasan por RPC internos protegidos con token y rate limiting.
- El historial válido se carga desde Supabase; el cliente no puede inyectarlo.
- WhatsApp valida la firma de Meta cuando WHATSAPP_APP_SECRET está configurado.
- Los eventos de WhatsApp tienen idempotencia persistida.
- Cada negocio puede guardar su WhatsApp Phone Number ID.
- La agenda es opcional y evita citas activas duplicadas.
- Los comprobantes se guardan en un bucket privado.

VARIABLES IMPORTANTES
Consultar .env.example. En producción son especialmente necesarias:
  INTERNAL_API_TOKEN
  WHATSAPP_VERIFY_TOKEN
  WHATSAPP_APP_SECRET
  WHATSAPP_ACCESS_TOKEN
  WHATSAPP_PHONE_NUMBER_ID
  NEXT_PUBLIC_SUPABASE_URL
  NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY

El hash SHA-256 de INTERNAL_API_TOKEN debe existir en private.app_secrets con la clave
internal_api_token. En el proyecto actual ya quedó configurado.

IA
AI_PROVIDER acepta deepseek, openai o none. La integración actual usa DeepSeek mediante
la API compatible con OpenAI. Si el proveedor falla o no tiene llave, se usa el fallback
determinista.
