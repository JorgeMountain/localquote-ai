Fecha: 2026-07-01

RESUMEN
Tactio es un MVP SaaS para negocios locales. Permite configurar negocios, cargar FAQs,
capturar leads desde un chat publico, guardar conversaciones en Supabase, crear solicitudes
de cita, crear cotizaciones estimadas y gestionar estados desde un dashboard privado.

Si trabajas desde otro PC con Codex, si: puedes continuar clonando el repositorio. Para
desarrollo normal solo necesitas Node.js, npm install y ejecutar el proyecto. Para deploy,
DB admin o cambios en Supabase/Vercel necesitas autenticar GitHub/Vercel/Supabase en ese PC.

FUNCIONALIDAD YA IMPLEMENTADA
- Auth con Supabase.
- Dashboard protegido por sesion.
- Negocios, FAQs, clientes, conversaciones, mensajes, citas y cotizaciones persistidos en Supabase.
- /businesses crea/edita negocios y FAQs reales.
- Chat publico guarda leads y mensajes.
- Deteccion de intencion: FAQ, cotizacion, cita, handoff.
- Cita se crea solo con datos minimos.
- Cotizacion se crea solo con datos minimos.
- Estados editables:
  - Leads: new, qualified, appointment, quoted
  - Citas: pending, confirmed, cancelled
  - Cotizaciones: draft, sent, accepted, rejected
- UX comercial:
  - Mensajes de exito/error en formularios.
  - Filtros por negocio en dashboard, conversaciones, citas y cotizaciones.
  - Vista detalle de conversacion/lead.
  - Botones rapidos: confirmar cita, enviar cotizacion, marcar aceptada.
- Marca visible actual: Tactio.

COSAS QUE FALTAN PARA MVP VENDIBLE
Orden recomendado:
1. Dominio y marca:
   - Comprar/configurar dominio para Tactio.
   - Cambiar alias de Vercel.
   - Crear logo simple y favicon.

2. Landing real:
   - Pagina publica que explique Tactio.
   - CTA: probar como cliente y entrar como dueno.
   - Demos por sector.

3. Onboarding:
   - Flujo guiado para crear primer negocio.
   - Evitar que parezca formulario tecnico.
   - Plantillas por sector: odontologo, tecnico, taller, estetica.

4. WhatsApp:
   - Integrar Twilio o WhatsApp Business Cloud API.
   - Reutilizar logica actual de /api/chat.
   - Guardar mensajes igual que el chat web.

5. Pagos:
   - Stripe o MercadoPago.
   - Plan simple mensual + prueba gratis.

6. Produccion/seguridad:
   - Quitar fallback hardcodeado de Supabase si se configura env real.
   - Revisar RLS/advisors.
   - Separar datos demo de datos reales.

7. Dashboard:
   - Busqueda por cliente/telefono.
   - Metricas por semana/mes.
   - Exportar leads CSV.
   - Notas internas por lead.

   - No usar service_role en cliente.
   - Si se modifica schema, actualizar migracion o crear una nueva migracion.

DECISIONES DE PRODUCTO TOMADAS
- Nombre actual: Tactio.
- No usar AI en la marca.
- Enfocarse en negocios locales, no enterprise.
- Inspiracion competitiva: empresas tipo Konecta trabajan CX enterprise; Tactio debe ser
  version simple, barata y lista para usar para negocios locales.
- OpenAI queda opcional hasta que haya monetizacion.
- WhatsApp es probablemente el siguiente canal mas vendible.
